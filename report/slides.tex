\section{Problem}


%%==========================================================================================
%%
\begin{slide}{Description}
  \begin{center}

    \twotonebox{\rotatebox{90}{description}}{\parbox{.86\textwidth}
    {
      The box office revenue of the movie is predicted by the given information. 
      \begin{itemize}
        \item 
        The data provided include actors, crew, plot keywords, budget, 
        poster, release date, language, production company, country, etc.   
        \item 
        Try to predict the total global box office revenue of the film.
      \end{itemize}
    }
    }

    \end{center}
    \bigskip
    \begin{center}
    \end{center}
    \bigskip
\end{slide}

\section{Conclusion}

%%==========================================================================================
%%
\begin{slide}[toc=,bm=]{Conclusion}
\begin{itemize}
\item The investment in the early stage and publicity in the later stage have an impact on the box office.
\smallskip

\item
\smallskip
In the early stage, the number of actors and crew should be moderate, not the more the better.

\item
\smallskip
The prediction accuracy can be further improved, such as using XGBoost
\end{itemize}
